# 문제: 14568 2017 연세대학교 프로그래밍 경시대회

## 문제

사탕 N개를 나누는 방식

- 1. 남는 사탕은 없음
- 2. 남규는 영훈이보다 2개 이상 많은 사탕을 가져야 함.
- 3. 셋 중 사탕을 0개 받는 사람은 없어야 함
- 4. 택희가 받는 사탕의 수는 홀수개가 되어서는 안됨

사탕을 세 사람에게 분배하는 서로 다른 경우의 수를 세보자.

## 풀이 - 완전탐색

방정식

- nam - young >= 2
- taek % 2 == 0
- nam > 0, young > 0, taek > 0

반복문

- nam in range(1, N)
- young in range(1, N - nam)
- taek = N - nam - young
